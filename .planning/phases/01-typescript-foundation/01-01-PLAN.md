  <files>tsconfig.json</files>
  <action>
Create tsconfig.json at project root with:
- "compilerOptions.module": "ESNext" (preserve ESM)
- "compilerOptions.target": "ES2022"
- "compilerOptions.strict": true (strict mode per TS-01)
- "compilerOptions.moduleResolution": "bundler" or "node"
- "compilerOptions.esModuleInterop": true
- "compilerOptions.allowJs": true (for gradual migration)
- "compilerOptions.checkJs": false (don't check existing JS files yet)
- "compilerOptions.outDir": "./dist"
- "compilerOptions.rootDir": "./src"
- "include": ["src/**/*"]
- "exclude": ["node_modules", "dist"]

This config enables TypeScript while allowing existing JS to work.
  </action>

  </verify>
